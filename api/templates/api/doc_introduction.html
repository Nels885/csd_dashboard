Visualisation des données pour la Calibration :

Request: GET

1. Méthode avec l'en-tête HTTP :

Headers:

Authorization: Token <TOKEN>

Url:

http://{{ website_domain }}/api/cal?search=<XELON_OR_VIN>

2. Méthode avec query string :

Url:

http://{{ website_domain }}/api/cal?search=<XELON_OR_VIN>&auth_token=<TOKEN>
